# Sales Calls Tracker

Enkel app for å tracke ringeaktivitet gjennom arbeidsdagen.

## Bruk

Åpne `index.html` i en nettleser. Ingen avhengigheter eller build-step.

## Funksjoner

- **4 handlingsknapper:** Ringt, Samtale, Oppfølging, Møte i kalender
- **Dagens metrics:** Viser antall samtaler, oppfølginger og møter (Ringt telles ikke i metrics)
- **Aktivitetslogg per time:** Viser aktiviteter for gjeldende klokketime
- **Nedteller:** 45 min arbeid + 15 min pause per time
- **localStorage:** Kun dagens data lagres, slettes automatisk ved ny dag
